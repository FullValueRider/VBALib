# VBALib

The code in VBALib has been created to support my Kvp Class and dabbling in AdventOfCode problems

The code as presented is a Work in Progress.  

The objective of code in VBALib (and also in the Kvp class) is to remove, as far as is practicable, boilerplate code from my AdventOfCode solutions.

Any reader/user of this code should be aware that I am not a professional programmer and that I code purely for 'fun'.

I welcome any comments, pointers, advice that allows me to do things 'betterer'
